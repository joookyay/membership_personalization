# 🛍️ 소비 유형별 개인화 마케팅

## 💡프로젝트 소개
- 멤버십 카드 사용 활성화를 위해 이욕고객을 군집화하여 군집 특성별로 마케팅 제안
- 연관분석, 요인분석을 활용한 상품 추천 

## ⚙️ 개발환경
언어: Python<br>
Tool: Jupyter Notebook
